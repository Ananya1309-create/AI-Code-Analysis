AI Code Analysis
This project compares AI-generated code with human-written code.

Objective
To analyze correctness, efficiency, and reliability of AI-generated programs.

Files
human_code.cpp → Optimized human-written code
ai_code.cpp → AI-generated version
performance.py → Basic time comparison
analysis.txt → Observations and conclusions
Conclusion
AI-generated code is helpful but requires validation and optimization for real-world use.
